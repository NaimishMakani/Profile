# 🌐 Personal Portfolio

This repository contains the source code for my personal portfolio website. It is designed to showcase my skills, projects, and experience through a modern, responsive, and visually appealing interface. The portfolio highlights my journey as a developer and serves as a central place to explore my work.

## 🔗 Live Demo

👉 **Visit Portfolio:** https://naimishmakani.github.io/Profile/

---

## ✨ Features

- 👋 Professional landing page
- 👨‍💻 About Me section
- 💼 Projects showcase
- 🛠️ Skills section
- 📱 Fully responsive design
- 🎨 Clean and modern user interface
- ⚡ Fast and lightweight
- 🌐 Deployed using GitHub Pages

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3
- **Styling:** Bootstrap / Tailwind CSS
- **Deployment:** GitHub Pages

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/NaimishMakani/Profile.git
```

Navigate to the project directory:

```bash
cd Profile
```

Open `index.html` in your preferred web browser.
